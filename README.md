Use this method if you want to run Dart from your terminal or command prompt.

Download: Get the SDK from dart.dev/get-dart.

Extract: Unzip the downloaded folder.

Install: Copy the extracted SDK folder and paste it into your C:\ drive.

Copy Path: Open the Dart folder in your C:\ drive, go into the bin folder, and copy the full file path from the top address bar.

Environment Variables: * Search for "Environment Variables" in Windows.

Under "System Variables," find Path, click Edit, then click New.

Paste the path to the bin folder.

Verify: Open your terminal and type: dart --version to check if it is installed
